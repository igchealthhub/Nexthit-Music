# CLAUDE.md

## Stack
- React
- Vite
- Supabase
- Vercel

## Development Rules
- Run `npm run build` after every change.
- Test signup and login flows after authentication-related changes.
- Test song upload and approval workflows after music/admin changes.
- Verify new users are created in Supabase Auth when testing signup.
- Use Playwright to test changes automatically.
- Commit and push successful fixes.
- Never delete production data without explicit user approval.
